# Project Overview

This project simulates a real-world hospital management system and performs advanced SQL-based analytics to generate meaningful business insights.

The goal of this project is to analyze patient behavior, doctor performance, hospital revenue, and departmental efficiency using structured relational databases.

This project demonstrates strong SQL fundamentals along with advanced analytical querying techniques used in real-world data analyst roles.

# Database Schema

The database consists of the following tables:

patients

doctors

departments

appointments

treatments

bills

These tables are connected using primary and foreign keys to simulate real hospital operations.

# Key Business Problems Solved

🔹 Patient Analytics

Identify high-frequency patients

Calculate patient visit trends

Detect potential high-risk patients (multiple visits)

🔹 Doctor Performance Analysis

Rank doctors by patient load

Calculate total revenue generated per doctor

Identify top-performing doctors

🔹 Revenue Analysis

Monthly hospital revenue trend

Running total revenue using window functions

Revenue by department

🔹 Operational Insights

Most common diagnoses

Average treatment cost

Payment status distribution
